# Components

Here is the documentation for the UI components used in my portfolio! Thanks to the power of **Storybook**, we can view our components in isolation from our app's logic.
This makes it easy to configure and test all sorts of styles and states, which is great during the _development_, _testing_, and _QA_ phase.
